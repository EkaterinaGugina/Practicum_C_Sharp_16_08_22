** Cодержание  проектов

+ Ex25: Напишите цикл, который принимает на вход два числа (A и B) и возводит число A в натуральную степень B.

   >3, 5 -> 243 (3⁵)

    >82 -> 10

    >9012 -> 12

+ Ex29: Напишите программу, которая задаёт массив из 8 элементов и выводит их на экран.

    >6, 1, 33 -> [6, 1, 33]
